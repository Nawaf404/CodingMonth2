محليا Locally

انشاء مستودع

لكي تستطيع رفع ملفات ، نشر مشاريعك ستحتاج الى تجهيز مستودع محلي بكل بساطة اكتب الامر التالي

git init
للتأكد من اضافته بشكل صحيح اذهب الى الملفات المخفيه

اضافة الملف للمستودع

لرع الملف للمستودع وحفظه ومتابعة احداثه يجب عليك اضافته

git add "[file]"

وعند رغبتك في اضافة كل الملفات بضغطه وحده اكتب . بالنهايه

git add .
